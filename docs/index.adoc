== Overview

UTF8DOK is a universal documentation platform that combines:

* *Compliance Engine*: Bridge Framework ADR validation with one-click fixes
* *Content Intelligence*: Diagram syntax highlighting and writing quality suggestions
* *Developer Experience*: VS Code extension with real-time feedback
* *Team Visibility*: Compliance dashboards and metrics

== Quick Start

. Install the VS Code extension from the Marketplace
. Open any AsciiDoc document
. See real-time compliance suggestions
. Use Quick Fixes (lightbulb) to fix violations
