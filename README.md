# utf8dok

> Plain text, powerful docs

A blazing-fast document processor for UTF-8 text formats, written in Rust.

## Status

**Early Development** - This project is in its initial stages.

## Features (Planned)

- AsciiDoc parsing and conversion
- Multiple output formats (HTML, PDF, etc.)
- WebAssembly support for browser usage
- High performance for large documents
